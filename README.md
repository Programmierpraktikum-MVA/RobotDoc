# Installation und Startanleitung


1. **Repository Klonen und in aktuellen Branch wechseln**


   ```bash
   git clone https://github.com/Programmierpraktikum-MVA/RobotDoc.git
   git checkout main
   ```

2. Anwendung starten

	```bash
 	 chmod +x start-dev.sh
	 ./start-dev.sh
	```
